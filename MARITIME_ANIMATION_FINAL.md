## 🌊 تأثيرات الحركة الانسيابية

### 1. **الطبقة الأولى - حركة بيضاء سريعة** ⚪
```typescript
<motion.path
    d={path}
    stroke="#ffffff"
    strokeWidth={isMain ? 1.2 : 0.8}
    strokeOpacity={0.5}
    strokeDasharray="8 20"
    animate={{ strokeDashoffset: -200 }}
    transition={{
        duration: isMain ? 4 : 5,
        repeat: Infinity,
        ease: 'linear',
    }}
/>
```
**التأثير**: خطوط بيضاء تتحرك بسرعة على جميع المسارات

---

### 2. **الطبقة الثانية - توهج برتقالي (للمسارات الرئيسية فقط)** 🔥
```typescript
<motion.path
    d={path}
    stroke="#f97316"
    strokeWidth={1.5}
    strokeOpacity={0.3}
    strokeDasharray="12 30"
    animate={{ strokeDashoffset: -250 }}
    transition={{
        duration: 3.5,
        repeat: Infinity,
        ease: 'linear',
    }}
    filter="url(#routeGlow)"
/>
```
**التأثير**: توهج برتقالي يتحرك بسرعة مختلفة للمسارات الرئيسية

---

### 3. **الطبقة الثالثة - خط الأساس** 📍
```typescript
<motion.path
    d={path}
    stroke={routeColor}
    strokeWidth={isMain ? 2 : 1.2}
    initial={{ pathLength: 0 }}
    animate={{ pathLength: 1 }}
    transition={{ duration: 0.8, ease: 'easeOut' }}
/>
```
**التأثير**: الخط الأساسي الثابت الذي يظهر تدريجياً

## 🎯 الميزات الإضافية

### 1. **موانئ نابضة** 💫
```typescript
animate={{
    r: [size * 2, size * 4, size * 2],
    strokeOpacity: [0.5, 0, 0.5],
}}
transition={{
    duration: 2,
    repeat: Infinity,
}}
```

### 2. **أسهم اتجاهية** ➡️
- تظهر على المسارات الرئيسية
- تُحدد اتجاه حركة السفن
- تظهر تدريجياً مع المسار

### 3. **تأثيرات Hover** 🎯
- عند المرور على ميناء: يكبر ويتوهج
- عند المرور على مسار: يزداد السُمك والتوهج
- عرض معلومات تفصيلية

### 4. **Great Circle Routes** 🌍
- مسارات واقعية على سطح الأرض الكروي
- معالجة خط التاريخ الدولي (180°)
- أقصر المسافات بين الموانئ
