# 📸 ميزة صور الخدمات

## ✨ نظرة عامة

تم إضافة ميزة **رفع الصور للخدمات** بالكامل! الآن يمكنك إضافة صورة معبّرة لكل خدمة تظهر بشكل احترافي في الفرونت إند.

---

## 🎯 كيفية رفع صورة لخدمة

### من الداشبورد:

1. **افتح الداشبورد**
   ```
   /dashboard/website-settings → Services → Services
   ```

2. **اختر خدمة موجودة أو أضف جديدة**
   - للتعديل: انقر على ✏️ بجانب الخدمة
   - للإضافة: انقر على "خدمة جديدة"

3. **ستجد قسم "صورة الخدمة"**
   - مباشرة بعد اختيار الأيقونة
   - رسالة توضيحية: "صورة تعبر عن الخدمة، ستظهر في صفحة تفاصيل الخدمة"

4. **رفع الصورة**
   - انقر على منطقة الصورة
   - اختر صورة من جهازك
   - الصورة سيتم:
     - ✅ ضغطها تلقائياً (وفر مساحة)
     - ✅ تحويلها لـ WebP (أسرع تحميل)
     - ✅ رفعها إلى `/public/images/services/`

5. **احفظ التغييرات**
   - انقر على "حفظ"
   - الصورة ستظهر فوراً في الفرونت إند!

## 🖼️ أين تظهر الصورة في الفرونت إند؟

### 1️⃣ صفحة الخدمات الرئيسية (`/services`)

#### **عرض البطاقة:**
- إذا **كانت هناك صورة**:
  ```
  ┌─────────────────────────────┐
  │  [صورة كبيرة 100% العرض]    │ ← الصورة تملأ عرض البطاقة
  │     ارتفاع: 192 بكسل         │
  │  [الأيقونة في الزاوية]       │ ← الأيقونة في زاوية الصورة
  └─────────────────────────────┘
       العنوان
       الوصف المختصر
       المميزات
  ```

- إذا **لم تكن هناك صورة**:
  ```
  ┌──────┐
  │ أيقونة │ ← فقط الأيقونة (مربع صغير)
  └──────┘
   العنوان
   الوصف المختصر
   المميزات
  ```

#### **التأثيرات:**
- ✨ **Hover Effect**: الصورة تكبر قليلاً (scale 1.1)
- 🎨 **Gradient Overlay**: تدرج أسود من الأسفل للأعلى
- 💫 **Smooth Transition**: انتقال سلس (700ms)

### 2️⃣ صفحة تفاصيل الخدمة (`/services/[slug]`)

#### **قسم الصورة المميز:**
إذا كانت هناك صورة، يظهر **قسم كامل** بعد الـ Hero مباشرة:

## 🔧 التفاصيل التقنية

### نظام رفع الصور:

```typescript
// المسار في الداشبورد
app/dashboard/website-settings/components/tabs/services/ServicesManagementTab.tsx

// المكون المستخدم
<ImageUploader
    folder="services"              // المجلد: /public/images/services/
    currentImage={formData.image}  // الصورة الحالية
    onUpload={(path) => setFormData({ ...formData, image: path })}
    aspectRatio="video"            // النسبة 16:9
    isRTL={isRTL}
/>
```

### API Endpoint:

```
POST /api/upload
Body: FormData {
    file: File,
    folder: "services"
}

Response: {
    success: true,
    path: "/images/services/1704892800000-abc123.webp",
    filename: "1704892800000-abc123.webp",
    originalSize: 245760,
    optimizedSize: 98304,
    savings: "60%"
}
```

2. ✅ **اختر الأيقونة**
3. ✅ **ارفع الصورة**
4. ✅ **املأ العناوين والأوصاف**
5. ✅ **أضف المميزات**
6. ✅ **احفظ**
7. ✅ **تحقق من النتيجة**

## ⚠️ ملاحظات هامة

### ✅ DO:
- استخدم صوراً عالية الجودة
- التزم بالأبعاد الموصى بها (1200×600)
- استخدم صوراً معبّرة عن الخدمة
- تأكد من وضوح الصورة

### ❌ DON'T:
- لا ترفع صوراً صغيرة جداً (ستكون مشوشة)
- لا ترفع صوراً ضخمة جداً (>5MB)
- لا تستخدم صوراً غير واضحة
- لا تستخدم صوراً لا علاقة لها بالخدمة

---

## 🔄 تعديل أو حذف صورة

### لتغيير الصورة:
1. افتح الخدمة للتعديل
2. مرّر على الصورة الحالية
3. انقر على "تغيير" (Change)
4. اختر صورة جديدة
5. احفظ

### لحذف الصورة:
حالياً، لا يوجد زر "حذف" مباشر. ولكن:
- يمكنك رفع صورة "فارغة" أو "placeholder"
- أو اترك الحقل فارغاً عند الإنشاء
- الأيقونة ستظهر تلقائياً إذا لم تكن هناك صورة
